# PetShop
# 🐾 Pet Machine

Aplicação Java via terminal para simular o funcionamento de uma máquina de banho para pets.

## Sobre o projeto

Sistema desenvolvido em Java com execução via terminal, criado durante o bootcamp da [DIO](https://www.dio.me). O programa simula uma máquina de banho para pets, permitindo cadastrar o animal, abastecer a máquina com água e shampoo, dar banho e gerenciar todo o processo via menu interativo.

## Menu principal
===Escolha uma das opções===
1 - Dar banho no pet

2 - Abastecer a máquina com água

3 - Abastecer a máquina com shampoo

4 - Verificar água da máquina

5 - Verificar shampoo da máquina

6 - Verificar se tem pet no banho

7 - Colocar pet na máquina

8 - Retirar pet da máquina

9 - Limpar a máquina

0 - Sair

## Funcionalidades

- Cadastra um pet pelo nome e coloca na máquina
- Abastece a máquina com água e shampoo
- Verifica os níveis de água e shampoo disponíveis
- Inicia o banho do pet
- Verifica se há um pet na máquina
- Retira o pet e limpa a máquina

## Tecnologias

- Java
- Scanner (entrada de dados)
- Orientação a objetos
